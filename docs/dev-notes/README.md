## 작성 규칙

- 파일명: `YYYY-MM-DD_제목_개발노트.md`
- 문서 상단에 **작성일/저장소/작성자** 메타데이터
- 다이어그램은 Mermaid 코드블록 (GitHub · Notion 모두 렌더 지원)
- 코드 스니펫은 언어 명시 (` ```js ` / ` ```css ` 등)
- 아이콘은 단색 시스템에 맞춰 본문 중 추가 가능

## 새 노트 작성 시

직전 노트의 §9 "향후 작업" 항목을 점검하여 무엇이 처리되고 무엇이 이월됐는지 추적합니다.
