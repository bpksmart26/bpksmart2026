# 노션 DB 속성 목록 (39개)

> BPK Smart 2026 — 통합정보 ↔ 노션 동기화 설계
> 노션 DB를 만들 때 이 문서를 체크리스트로 사용하세요.

## ⚠️ 중요 규칙

- **속성 이름은 이 문서 그대로 입력** (띄어쓰기·언더스코어 1자라도 다르면 sync 실패)
- **`회사명`은 Title 속성** (DB 만들 때 기본 "이름" 컬럼을 rename)
- **양방향 ✅ 표시 4개**(신청상태/담당자/신청메모/견적메모)만 노션에서 수정 가능. 나머지는 수정해도 다음 sync 때 시트값으로 되돌아감
- **운영 컬럼은 자유롭게 추가** (우선순위/다음컨택일/태그 등) — sync는 절대 안 건드림

## Select / Multi-select 옵션 — 미리 만들어두면 좋은 것

옵션은 GAS가 값 보낼 때 자동으로 추가되지만, **색상을 본인 톤으로** 잡고 싶으면 미리 만들어두세요.

### 신청상태 (Select)
- `접수` — 파랑
- `검토중` — 노랑
- `견적준비` — 보라
- `견적발송` — 주황
- `수주` — 초록
- `보류` — 회색
- `종결` — 검정

### 견적상태 (Select)
- `작성중` — 회색
- `발급완료` — 초록
- `재발급` — 노랑

### 문제유형 (Select)
- (현재 신청폼에서 사용하는 옵션 그대로 — 코드 확인 후 채움)

### 에어유무 (Select)
- `있음`
- `없음`

### 공정 / 포장형태 / 문제점 / 추천장비 / 전기 (Multi-select)
- 미리 만들 필요 없음 (값이 들어올 때 자동 추가)
